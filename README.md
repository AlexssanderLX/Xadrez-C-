# Chess Game (C# Console)

Academic chess game developed in C# as part of a programming course, focusing on object-oriented programming and game logic.

## Features
- Playable chess game in console
- Board representation
- Piece movement logic
- Turn-based gameplay

## Technologies
- C#
- .NET Console Application

## Purpose
This project was created to practice:
- Object-Oriented Programming (OOP)
- Game logic and state control
- Code organization and readability

## How to Run
1. Clone the repository
2. Open the solution in Visual Studio
3. Run the project (`Ctrl + F5`)

## Notes
This is an academic project and may receive future improvements.
